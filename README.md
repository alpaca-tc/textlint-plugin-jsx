> JSX support for textlint

## Install

```sh
yarn add textlint-plugin-jsx
```

## Usage

```json
{
  "plugins": {
    "jsx": true
  }
}
```

```json
{
  "plugins": {
    "jsx": {
      "extensions": [".custom-ext"]
    }
  }
}
```
